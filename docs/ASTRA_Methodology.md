# 1. Core Model Structure  

ASTRA CORE™ evaluates Decision Integrity (D) as a function of five structural variables:

D_{i} = f(C_{i}, A_{i}, L_{i}, E_{i}, S_{i})

Where:

- **Cᵢ** = Cyber Stability  
- **Aᵢ** = AI Governance Control  
- **Lᵢ** = Legal Coherence  
- **Eᵢ** = Economic Resilience  
- **Sᵢ** = Social Stability  

Each variable represents a normalized score.

# 3. Normalization Framework  

All variables are normalized:

X_{norm} = \frac{X - X_{min}}{X_{max} - X_{min}}

Ensuring cross-system comparability.

---

# 4. Weighted Index Construction  

D_{i} = w_C C_{i} + w_A A_{i} + w_L L_{i} + w_E E_{i} + w_S S_{i}

### Default Weights:

- wC = 0.25  
- wA = 0.20  
- wL = 0.20  
- wE = 0.20  
- wS = 0.15  

---

## 4.1 Weight Justification  

The weighting structure reflects systemic priority:

- Cyber Stability (0.25) — critical infrastructure dependency  
- AI Governance (0.20) — emerging control layer  
- Legal Coherence (0.20) — institutional backbone  
- Economic Resilience (0.20) — financial stability  
- Social Stability (0.15) — outcome stability  

Weights are calibrated based on systemic influence and risk propagation.
